Python 虛擬環境

環境設置基本上直接問GPT，比較不一樣的是，我大學寫 python 有用虛擬環境大部分是用 conda ，我今天才知道可以用 python 直接內建一個。

1️⃣ python -m venv venv

作用:

這條指令的目的是 建立一個新的虛擬環境，它會在當前目錄下建立一個名為 venv 的資料夾。

拆解:

  • python：調用 Python 解釋器
  • m venv：使用 Python 內建的 venv 模組來建立虛擬環境
  • venv：指定虛擬環境的名稱（這裡我們取名為 venv，但你可以改成其他名稱，如 fastapi-env）

2️⃣ source venv/bin/activate（macOS / Linux）

作用

這條指令的目的是 啟動虛擬環境，讓你的終端機開始使用 venv 內的 Python 和套件，而不是系統全局的 Python。

拆解

  • source：這是一個 shell 指令，負責執行 venv/bin/activate 腳本，並讓環境變數生效
  • venv/bin/activate：這個腳本會修改你的終端機環境，讓 Python 和 pip 指向 venv 內的版本

💡 總結

指令 作用
python -m venv venv 建立 虛擬環境
source venv/bin/activate 啟動 虛擬環境（macOS/Linux）
venv\Scripts\activate 啟動 虛擬環境（Windows）
deactivate 退出 虛擬環境
